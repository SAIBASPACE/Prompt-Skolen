## Quick reference

### Gratis version - Bedst til:
- Email-skrivning
- Oversættelse
- Opsummering af korte tekster
- Simpel kode

### Plus features:
- **Hukommelse:** Gem kontekst, spar 30-60 sek per prompt
- **Custom GPTs:** Specialiserede assistenter til gentagne opgaver
- **Fil-upload:** Analysér PDF, Word, Excel direkte
- **Browsing:** Aktuel information med kilder
- **Code Interpreter:** Data-visualisering og beregninger

### Hvornår opgradere til Plus?

✓ Du bruger ChatGPT dagligt
✓ Du arbejder med dokumenter
✓ Du laver research
✓ Dit team skal bruge samme prompts

## Praktisk opsætning (kun Plus)

### Hukommelse

```
Husk: Jeg hedder [navn], arbejder som [rolle] hos [firma].
Vi bruger [værktøjer] og har [antal] ansatte.
Vores tone er [beskrivelse].
```

### Tjek hukommelse

```
Hvad husker du om mig?
```

### Fil-analyse

Upload fil og skriv:
```
Opsummer dette dokument i 5 bullet points.
Fremhæv vigtigste tal og konklusioner.
```

## Sikkerhedspåmindelse

**Må ALDRIG indtastes i ChatGPT gratis:**
- CPR-numre, kundenavne, adresser
- Passwords, API-nøgler
- Uoffentliggjorte regnskaber, strategier, kontrakter

ChatGPT Plus (og Enterprise) har databeskyttelse - men tjek altid jeres virksomheds politik først.
