# Microsoft Copilot: Basics

**Tid at lære:** 6 minutter
**Virker i:** Microsoft Copilot (gratis + Microsoft 365)

---

## Hvad er Copilot?

Microsoft Copilot er AI integreret direkte i Word, Excel, Outlook, Teams og PowerPoint.

Hvis din virksomhed bruger Microsoft 365, er Copilot den nemmeste vej til AI i hverdagen - du skal ikke skifte værktøj.

---

## Gratis vs Microsoft 365

| Feature | Gratis (copilot.microsoft.com) | Microsoft 365 |
|---------|-------------------------------|---------------|
| Chat | ✓ Ja | ✓ Ja |
| Billeder (DALL-E) | ✓ Ja | ✓ Ja |
| Word integration | ✗ | ✓ Ja |
| Excel integration | ✗ | ✓ Ja |
| Outlook integration | ✗ | ✓ Ja |
| Teams integration | ✗ | ✓ Ja |
| PowerPoint integration | ✗ | ✓ Ja |

**Gratis version** fungerer som selvstændig chatbot - lignende ChatGPT.

**Microsoft 365 Copilot** koster ekstra (ca. 280 kr/md per bruger) og kræver Microsoft 365 Business Standard eller højere.

---

## Hvornår Copilot vs andre værktøjer?

| Brug | Copilot | Bedre alternativ |
|------|---------|------------------|
| Office-dokumenter | ✓ | - |
| Mødeopsamling (Teams) | ✓ | Otter.ai (hvis ikke M365) |
| Generel chat | Ja | ChatGPT (bedre) |
| Kode | ✗ | GitHub Copilot |
| Research | ✗ | Perplexity |
| Lange dokumenter | Word | Claude (bedre kontekst) |

**Brug Copilot hvis:** Du arbejder primært i Microsoft Office.

**Brug ChatGPT/Claude hvis:** Du ikke har Microsoft 365 eller vil have bedre chat-funktionalitet.

---

## SIKT Framework til Copilot

For at få gode resultater med Copilot i Office, brug SIKT-modellen:

- **S:** Specificér hvilken app (Word, Excel, Outlook, Teams, PowerPoint)
- **I:** Instruktion - hvad skal Copilot gøre præcist
- **K:** Kontekst - baggrund, data, situation
- **T:** Tone og format - stil, længde, struktur

---

## Sammenligning

```
Lav en præsentation om Q3
```

```
Lav en præsentation om vores Q3-resultater.

Indhold:
- Omsætning: 12.4 mio (op 15%)
- Nye kunder: 47
- Største udfordring: leveringstider

Stil: Professionel, minimal
Antal slides: 6-8
Inkluder: Grafer hvor relevant
```

**Dårlig prompt:** Copilot opfinder tal og indhold. Du bruger mere tid på at rette end du sparede.

**God prompt:** Copilot laver struktur og layout med korrekte data.

---

## Sådan starter du med Copilot

### I gratis version

1. Gå til **copilot.microsoft.com**
2. Log ind med din Microsoft-konto
3. Brug det som en chatbot (lignende ChatGPT)

### I Microsoft 365

1. Åbn Word, Excel, Outlook, Teams eller PowerPoint
2. Klik på **Copilot-ikonet** (øverst til højre eller i menu)
3. Skriv din prompt i Copilot-panelet

---

## Gevinster ved Copilot

Med Copilot i Microsoft 365 får du:

- **5 min** sparet per mødeopsamling (Teams)
- **10 min** sparet per rapport (Word)
- **80%** hurtigere email-svar (Outlook)
- **15 min** sparet på dataanalyse (Excel)

**Men:** Copilot er bedst når du allerede arbejder i Microsoft 365. Skift ikke workflow bare for at bruge Copilot.

---

## Begrænsninger

| Begrænsning | Workaround |
|-------------|------------|
| Kræver Microsoft 365 licens | Brug gratis version til simpel chat |
| Excel kræver tabelformat | Konverter data med Ctrl+T først |
| PowerPoint opfinder fakta | Tjek alle tal manuelt |
| Teams kræver transkription | Slå transkription til før møder |
| Kan ikke læse password-beskyttede filer | Fjern beskyttelse midlertidigt |

## Øvelse (3 minutter)

**Gratis version:**

Gå til copilot.microsoft.com og test en simpel prompt:

**Copyable:**
```
Skriv en email til en kunde der spørger om leveringstid.
Svar: Vi sender i morgen, levering onsdag.
Tone: Professionel men venlig.
Max 60 ord.
```

Sammenlign svaret med ChatGPT hvis du har adgang. Hvad er forskellen?

**Microsoft 365:**

Åbn et Office-program du bruger dagligt og find Copilot-ikonet. Eksperimentér med at stille et spørgsmål.
