## Quick reference

### Gratis vs Microsoft 365

| Feature | Gratis (copilot.microsoft.com) | Microsoft 365 |
|---------|-------------------------------|---------------|
| Chat | ✓ | ✓ |
| Billeder (DALL-E) | ✓ | ✓ |
| Word integration | ✗ | ✓ |
| Excel integration | ✗ | ✓ |
| Outlook integration | ✗ | ✓ |
| Teams integration | ✗ | ✓ |
| PowerPoint integration | ✗ | ✓ |

### SIKT Framework til Copilot

**S - Specificér app** (Word, Excel, Outlook, Teams, PowerPoint)
**I - Instruktion** (Hvad skal Copilot gøre)
**K - Kontekst** (Baggrund, data, situation)
**T - Tone og format** (Stil, længde, struktur)

### Eksempel-prompts

**Word - Email:**
```
Skriv en email til kunde om forsinket levering.
Tone: Professionel og undskyldende
Årsag: Leverandørforsinkelse
Ny dato: 3 dage ekstra
Tilbyd: 10% rabat på næste ordre
Max 80 ord
```

**Excel - Data-analyse:**
```
Analysér salgsdata i kolonne B-D (januar-marts).
Find top 3 produkter efter omsætning.
Beregn vækst fra januar til marts i procent.
Vis resultat som søjlediagram.
```

**Outlook - Email-svar:**
```
Svar på denne email.
Tone: Venlig men professionel
Svar: Ja til mødet tirsdag kl. 14
Tilføj: Bed om dagsorden på forhånd
Max 50 ord
```

**Teams - Mødeopsamling:**
```
Opsummer mødet i 5 punkter.
Fremhæv beslutninger og action items.
Angiv hvem der er ansvarlig for hver action.
Format: Punktliste med navne
```

**PowerPoint - Præsentation:**
```
Lav præsentation om Q3-resultater.
Omsætning: 12.4 mio (op 15%)
Nye kunder: 47
Største udfordring: leveringstider
Stil: Professionel, minimal
Slides: 6-8
```

## Vigtige begrænsninger

| Begrænsning | Workaround |
|-------------|------------|
| Kræver Microsoft 365 licens | Brug gratis version til simpel chat |
| Excel kræver tabelformat (Ctrl+T) | Konverter data først |
| PowerPoint opfinder fakta | Tjek alle tal manuelt |
| Teams kræver transkription | Slå transkription til før møder |

---

## Hvornår Copilot vs andre værktøjer?

| Opgave | Copilot | Bedre alternativ |
|--------|---------|------------------|
| Office-dokumenter | ✓ | - |
| Mødeopsamling | ✓ (Teams) | Otter.ai |
| Generel chat | Ja | ChatGPT (bedre) |
| Kode | Nej | GitHub Copilot |
| Research | Nej | Perplexity |
| Lange dokumenter | Word | Claude (større kontekst) |
